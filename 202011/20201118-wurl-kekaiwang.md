# Wurl



## 简介
Wurl将同时检查每个链接的HTTP / S状态代码以及状态代码列表。当从waybackmachine获得很多链接并且想要稍微整理一下列表时，该功能很有用。Wurl有多个选项，可让您的生活更轻松。
## 安装
    wurl的安装非常简单，只要电脑上有go环境直接使用 go get 命令就可以获取。

```
go get -u github.com/bp0lr/wurl
```
